Esta tela pode ajudá-lo a testar a conectividade de seu data source. Testar a conexão verifica se uma conexão pode ser estabelecida e exibe algumas informações do banco de dados. Executar SQL executa uma query SQL contra o banco de dados. Você pode ver a saída da operação ou uma mensagem de erro na parte inferior da tela. Exibir histórico exibe uma lista com as queries que você executou. Por favor, considere usar Exibir opções para limitar o número de registros retornados pelas suas queries para não exaurir os recursos do sistema.
