Cet écran permet de tester vos sources de données. Tester connexion vérifie si la connexion peut être établie et affiche des infos sur la base. Exécuter SQL exécute la requête SQL saisie. Voir le message en bas de l'écran ainsi que le résultat ou les données issues de l'exécution. Avec l'historique affiche la liste des requêtes déjà exécutées. Utiliser Avec les options pour limiter le nombre de lignes renvoyées par votre requête afin de limiter la consommation de ressources.
