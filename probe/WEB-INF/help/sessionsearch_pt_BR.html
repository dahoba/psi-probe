Os seguintes critérios de busca estão disponíveis para filtrar a lista de sessões. O ID da sessão satisfaz uma expressão regular. O Último IP contém uma string. O Tempo de inatividade está dentro de um intervalo especificado em segundos. A Idade está dentro de um intervalo especificado em segundos. Os Nomes de atributos satisfazem pelo menos uma expressão regular de uma lista de expressões regulares separadas por vírgula. Este critério permite encontrar uma sessão com um atributo A e/ou um atributo B.
